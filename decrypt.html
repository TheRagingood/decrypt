Important Note

On Monday, May 1, 2017, the Dropbox extension as well as the Google Drive extension were updated to include a missing parameter from the archive files. This parameter is required in order to convert your password into your encryption key. If you configured your sync adapter before this date, please perform these steps to update your files:

  1. Open your Standard File dashboard, and navigate to "Sync Adapters".
  2. Press "Perform Initial Sync" on relevant adapter.

Information

Use this offline tool to decrypt an encrypted Standard Notes backup file. You can select more than one file in the file chooser.

The result will be a file that is downloaded that contains the contents of your items in their decrypted form.
